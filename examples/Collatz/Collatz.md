# Collatz

Write a program called Collatz.

1. Prompt the user for an integer.
2. If the number equals 1, you are done.
3. Print the number.
4. If the number is divisible by 2, divide it by 2.
5. Otherwise, multiply it by 3 then add 1.
6. Go back to step 2.

## Test Input
6

## Expected Output
6
3
10
5
16
8
4
2
1
