# FizzBuzzHundred

Write a program called FizzBuzzHundred that iterates through every integer from 1 to 100.

For each number:

- If the number is divisible by both 7 and 11, print **FizzBuzz**.
- If the number is divisible by 7 (but not 11), print **Fizz**.
- If the number is divisible by 11 (but not 7), print **Buzz**.
- Otherwise, print the number itself.

The loop counter is an integer variable named `n`.
