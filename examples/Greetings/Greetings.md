# Greetings

Write a program called Greetings.

Print the message "Hello! What is your name?".
Read the user's name from standard input into a string variable called user_name.
Print the message "Nice to meet you, " followed by the value of user_name.

## Test Input

Alice

## Expected Output

```
Hello! What is your name?
Nice to meet you, {user_name}
```
