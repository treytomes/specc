# FizzBuzz

Write a program called FizzBuzz that iterates through every integer from 1 to 100.

For each number:

- If the number is divisible by both 3 and 5, print **FizzBuzz**.
- If the number is divisible by 3 (but not 5), print **Fizz**.
- If the number is divisible by 5 (but not 3), print **Buzz**.
- Otherwise, print the number itself.

The loop counter is an integer variable named `n`.
